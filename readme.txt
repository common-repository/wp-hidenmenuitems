== Description ==

WP-HidenMenuItems is a WordPress CMS plugin that provides hides menu items, pages or posts for all but selected site visitors. The plugin is simple, like a Kalashnikov assault rifle. The plugin is managed in the admin panel: Appearance -> Hide menu items.
Recommendation: to improve the quality of the interface, install the plugin <a href="https://wordpress.org/plugins/watchman-site7/" target="_blank">WatchMan-Site7</a>

<a href="https://adminkov.bcr.by/contact/" target="_blank">Plugin home page</a>

You can send a letter to the developer at: klenitskiy.oleg@mail.ru

== Installation ==

Plugin settings are in the admin panel: Appearance -> Hiden menu items.
